# Screenshot Saver (Chrome Extension)

キャプチャ結果を新規タブの保存ページでプレビューし、PNG/JPG/PDFとして保存、またはクリップボードへコピーできるChrome拡張です。

## セットアップ

1. このリポジトリをローカルに配置します。
2. Chromeで `chrome://extensions` を開きます。
3. 右上の「デベロッパーモード」を有効にします。
4. 「パッケージ化されていない拡張機能を読み込む」を押し、このフォルダを選択します。

## 使い方

1. 拡張アイコンをクリックしてポップアップを開きます。
2. キャプチャ種別を選択します。
   - 表示範囲: 現在のビューポートを保存します。
   - ページ全体: 自動スクロールして全体を合成します。
3. 新規タブで保存ページが開き、プレビューが表示されます。
4. 右側のパネルから以下を選択して保存します。
   - 画像として保存（PNG/JPG）
   - PDFとして保存（縦長は複数ページに分割）
   - クリップボードにコピー

## 仕様メモ

- キャプチャデータは `chrome.storage.session` に一時保存され、保存後に削除されます。
- PDFはA4縦ベースで複数ページに分割して出力します。
- ページ全体キャプチャは自動スクロールを行います。動的コンテンツが多いページでは、途中で描画が変わる場合があります。

## 制約

- `chrome://` など拡張がアクセスできないページはキャプチャできません。
- 長尺ページはメモリ制限で失敗する場合があります。
